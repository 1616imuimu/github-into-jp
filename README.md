# 「github-into-jp」について
## これは何？
GitHubのサイトを日本語に翻訳するChromeの拡張機能です。  
ソースコードはそのままに、メニューやメッセージだけを翻訳することを目指して作成しています。

## どうやって使うの？
プログラムをダウンロードし、Chromeに適用してください。  
適用の仕方は「パッケージ化されていない拡張機能を読み込む」で検索してください。  
分かりやすく説明してくれているサイトがいくつかあると思います。


<hr>

## 注意事項
- 翻訳の内容については十分に精査したうえで採用しておりますが、その内容の正確性などを保証するものではありません。
- いかなる場合でも、この機能を利用したために被った損害、損失に対して一切の責任を負いません。
- 翻の訳内容に疑問や不安を持たれた場合は、機能をオフにし原文にあたることをおすすめします。
